# Aashutosh Solanki Portfolio

A modern, responsive portfolio website built with HTML, CSS, and JavaScript featuring advanced animations and unique styling.

## Features

- **Modern Design**: Clean, professional layout with gradient accents
- **Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Navigation**: Smooth page transitions with active state indicators
- **Dark Mode Toggle**: Switch between light and dark themes
- **Animated Elements**: Typing effects, parallax scrolling, and fade-in animations
- **Download Resume**: Functional resume download button
- **Smooth Scrolling**: Seamless navigation between sections

## File Structure

```
portfolio/
├── index.html          # Main HTML structure
├── styles.css          # Complete styling with animations
├── script.js           # Interactive JavaScript functionality
└── README.md           # This file
```

## Customization Guide

### Profile Information
Update the following in `index.html`:
- **Name**: Change "Aashutosh Solanki" in multiple places
- **Title**: Update "Product Engineer | Backend Enthusiast"
- **Description**: Modify the hero section description
- **Profile Image**: Replace the placeholder image URL

### Content Sections
Each section (`about`, `skills`, `projects`, `contact`) is ready for your content:
- **About**: Add your background, education, and journey
- **Skills**: Create a grid of your technical skills
- **Projects**: Showcase your portfolio projects
- **Contact**: Add your contact information and form

### Styling Customization
In `styles.css`, you can modify:
- **Colors**: Update CSS variables in `:root`
- **Fonts**: Change font families in the `@import` rules
- **Animations**: Adjust timing and effects
- **Layout**: Modify responsive breakpoints

### Advanced Features
The portfolio includes:
- **Typing Animation**: Name types out automatically
- **Parallax Scrolling**: Hero section moves at different speed
- **Intersection Observer**: Elements animate when scrolled into view
- **Gradient Animations**: Rotating profile ring effect
- **Hover States**: Interactive buttons and navigation

## How to Use

1. **Open the portfolio**: Simply open `index.html` in your web browser
2. **Navigate**: Use the sidebar to switch between sections
3. **Customize**: Edit the files to add your personal content
4. **Deploy**: Upload to any web hosting service
